# Cacao & Alma - Chocolatería Artesanal

## Descripción

Este proyecto es un sitio web estático para **Cacao & Alma**, una chocolatería artesanal de lujo. El sitio presenta productos de chocolate, la historia de la chocolatería y opciones de contacto. Está diseñado con un enfoque en la elegancia y la experiencia del usuario.

## Tecnologías Utilizadas

- **HTML5**: Estructura del sitio web.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Funcionalidad interactiva, como el carrito de compras.
- **Fuentes de Google Fonts**: Playfair Display, Cormorant Garamond y Jost para tipografía elegante.

## Instalación y Uso

1. Clona o descarga el repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Navega por las secciones: Inicio, Productos, Historia y Contacto.

No se requieren dependencias adicionales ni un servidor local, ya que es un sitio web estático.

## Estructura del Proyecto

- `index.html`: Página principal del sitio web.
- `styles.css`: Archivo de estilos CSS para el diseño.
- `main.js`: Archivo JavaScript para la funcionalidad interactiva.
- `imagenes/`: Carpeta que contiene las imágenes utilizadas en el sitio.

## Características

- Navegación suave entre secciones.
- Carrito de compras básico con contador.
- Diseño responsivo para dispositivos móviles y de escritorio.
- Elementos visuales elegantes con SVG para decoraciones.
